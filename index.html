          🛡 Red Flags of Phishing Pages

          • URL doesn't match the real site
          • Padlock icon missing or broken
          • Urgent message asking for credentials
          • Generic branding or broken images
          • Unusual fonts or layout errors

          Your Security Toolkit

          Quick reference cards for everyday digital safety.

          🔑

          Password Tips

          • 12+ characters minimum
          • Mix letters, numbers, symbols
          • Never reuse passwords
          • Use a password manager
          ✉️

          Email Safety

          • Check sender's real email
          • Hover links before clicking
          • No bank asks for passwords
          • Report suspicious emails
          📡

          WiFi Security

          • Avoid public WiFi for banking
          • Use a VPN when traveling
          • Prefer WPA3 networks
          • Forget unused networks
          🖥️

          Device Hygiene

          • Enable 2FA everywhere
          • Keep software updated
          • Lock screen after 30 secs
          • Back up data regularly
